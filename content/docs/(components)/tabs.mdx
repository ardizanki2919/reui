## API Reference

This component is built using [Radix UI Tabs](https://www.radix-ui.com/primitives/docs/components/tabs) primitives. For detailed information, please visit the full API reference.

### Tabs

This component is based on the [Tabs.Root](https://www.radix-ui.com/primitives/docs/components/tabs#root) primitive. For full details, visit the official documentation.

### TabsList

This component is based on the [Tabs.List](https://www.radix-ui.com/primitives/docs/components/tabs#list) primitive and includes the following custom props:

| **Prop**                                                                            | **Type**                                                            | **Default** |
| ----------------------------------------------------------------------------------- | ------------------------------------------------------------------- | ----------- |
| `variant` <PopoverInfo>The visual style variant of the tabs list.</PopoverInfo>     | ~enum~ <PopoverInfo> "default" \| "button" \| "line" </PopoverInfo> | ~"default"~ |
| `size` <PopoverInfo>The size of the tabs list.</PopoverInfo>                        | ~enum~ <PopoverInfo> "lg" \| "md" \| "sm" \| "xs" </PopoverInfo>    | ~"md"~      |
| `className` <PopoverInfo>Additional class names for styling the list.</PopoverInfo> | ~string~                                                            | ~undefined~ |

### TabsTrigger

This component is based on the [Tabs.Trigger](https://www.radix-ui.com/primitives/docs/components/tabs#trigger) primitive and inherits the following props from context:

| **Prop**                                                                               | **Type**                                                            | **Default** |
| -------------------------------------------------------------------------------------- | ------------------------------------------------------------------- | ----------- |
| `variant` <PopoverInfo>Inherited from TabsList context.</PopoverInfo>                  | ~enum~ <PopoverInfo> "default" \| "button" \| "line" </PopoverInfo> | ~"default"~ |
| `size` <PopoverInfo>Inherited from TabsList context.</PopoverInfo>                     | ~enum~ <PopoverInfo> "lg" \| "md" \| "sm" \| "xs" </PopoverInfo>    | ~"md"~      |
| `className` <PopoverInfo>Additional class names for styling the trigger.</PopoverInfo> | ~string~                                                            | ~undefined~ |

### TabsContent

This component is based on the [Tabs.Content](https://www.radix-ui.com/primitives/docs/components/tabs#content) primitive and includes the following custom props:

| **Prop**                                                                               | **Type**                                      | **Default** |
| -------------------------------------------------------------------------------------- | --------------------------------------------- | ----------- |
| `variant` <PopoverInfo>The visual style variant of the content.</PopoverInfo>          | ~enum~ <PopoverInfo> "default" </PopoverInfo> | ~"default"~ |
| `className` <PopoverInfo>Additional class names for styling the content.</PopoverInfo> | ~string~                                      | ~undefined~ |
