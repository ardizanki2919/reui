## API Reference

This component provides a flexible way to display inline code snippets with optional copy functionality.

### Code

| **Prop**                                                                                              | **Type**                                                                    | **Default** |
| ----------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------- | ----------- |
| `variant` <PopoverInfo>Specifies the code's theme variant.</PopoverInfo>                              | ~enum~ <PopoverInfo> "default" \| "destructive" \| "outline" </PopoverInfo> | `default`   |
| `size` <PopoverInfo>Sets the size of the code component.</PopoverInfo>                                | ~enum~ <PopoverInfo> "sm" \| "default" \| "lg" </PopoverInfo>               | `default`   |
| `showCopyButton` <PopoverInfo>Whether to show a copy button next to the code.</PopoverInfo>           | ~boolean~                                                                   | `false`     |
| `copyText` <PopoverInfo>Custom text to copy when the copy button is clicked.</PopoverInfo>            | ~string~                                                                    | <Na/>       |
| `asChild` <PopoverInfo>Renders the code as a child element, using the `Slot` component.</PopoverInfo> | ~boolean~                                                                   | `false`     |
| `className` <PopoverInfo>Additional class names for styling the code component.</PopoverInfo>         | ~string~                                                                    | <Na/>       |
