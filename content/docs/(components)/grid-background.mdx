## API Reference

This component is built using [Motion](https://motion.dev/) primitives and includes the following custom props:

### GridBackground

| **Prop**                                                                                    | **Type**                                                                                                                            | **Default**                                                                                                                                                                                         |
| ------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `children` <PopoverInfo>Content to display over the grid background.</PopoverInfo>          | ~ReactNode~                                                                                                                         | <Na/>                                                                                                                                                                                               |
| `gridSize` <PopoverInfo>Grid dimensions in format 'cols:rows'.</PopoverInfo>                | ~enum~ <PopoverInfo> "4:4" \| "5:5" \| "6:6" \| "6:8" \| "8:8" \| "8:12" \| "10:10" \| "12:12" \| "12:16" \| "16:16" </PopoverInfo> | ~"8:8"~                                                                                                                                                                                             |
| `className` <PopoverInfo>CSS class for the background container.</PopoverInfo>              | ~string~                                                                                                                            | <Na/>                                                                                                                                                                                               |
| `colors.background` <PopoverInfo>Background color CSS class.</PopoverInfo>                  | ~string~                                                                                                                            | ~"bg-slate-900"~                                                                                                                                                                                    |
| `colors.borderColor` <PopoverInfo>Grid border color CSS class.</PopoverInfo>                | ~string~                                                                                                                            | ~"border-slate-700/50"~                                                                                                                                                                             |
| `colors.borderSize` <PopoverInfo>Grid border width in CSS units.</PopoverInfo>              | ~string~                                                                                                                            | ~"1px"~                                                                                                                                                                                             |

| `beams.shadow` <PopoverInfo>Shadow and styling CSS classes for beams.</PopoverInfo>         | ~string~                                                                                                                            | ~"shadow-lg shadow-cyan-400/50 rounded-full"~                                                                                                                                                       |
| `beams.speed` <PopoverInfo>Base animation speed in seconds for beam movement.</PopoverInfo> | ~number~                                                                                                                            | ~4~                                                                                                                                                                                                 |
