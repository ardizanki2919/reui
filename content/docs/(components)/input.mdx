## API Reference

This component extends the native HTML `<input>` element with additional styling and size variants.

### Input

This component is a custom implementation that extends the native HTML `<input>` element with the following custom props:

| **Prop**                                                                                   | **Type**                                                 | **Default** |
| ------------------------------------------------------------------------------------------ | -------------------------------------------------------- | ----------- |
| `variant` <PopoverInfo>Controls the variant of the input field.</PopoverInfo>              | ~enum~ <PopoverInfo> "md" \| "sm" \| "xs" </PopoverInfo> | ~"md"~      |
| `className` <PopoverInfo>Additional class names for styling the input field.</PopoverInfo> | ~string~                                                 | <Na/>       |
