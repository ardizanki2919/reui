## API Reference

The Alert component is a custom implementation that provides flexible and customizable alert messages. Here's the complete API reference:

### Alert

The root Alert component includes the following props:

| **Prop**                                                                                 | **Type**                                                                                                                  | **Default** |
| ---------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------- | ----------- |
| `variant` <PopoverInfo>Specifies the alert's theme.</PopoverInfo>                        | ~enum~ <PopoverInfo> "default" \| "primary" \| "destructive" \| "success" \| "info" \| "mono" \| "warning" </PopoverInfo> | ~default~   |
| `appearance` <PopoverInfo>Determines the appearance of the alert.</PopoverInfo>          | ~enum~ <PopoverInfo> "solid" \| "outline" \| "light" \| "stroke" </PopoverInfo>                                           | ~solid~     |
| `size` <PopoverInfo>Determines the size of the alert.</PopoverInfo>                      | ~enum~ <PopoverInfo> "xs" \| "sm" \| "md" </PopoverInfo>                                                                  | ~md~        |
| `className` <PopoverInfo>Additional class names for styling the alert.</PopoverInfo>     | ~string~                                                                                                                  | <Na/>       |
| `close` <PopoverInfo>Controls whether the alert can be closed by the user.</PopoverInfo> | ~boolean~                                                                                                                 | ~true~      |
| `onClose` <PopoverInfo>Callback function when the alert is closed.</PopoverInfo>         | ~function~                                                                                                                | <Na/>       |
