## API Reference

### TypingText

This component is built using [Motion](https://motion.dev/) primitives and includes the following custom props:

| **Prop**                                                                                            | **Type**                                                                                                                                                                      | **Default** |
| --------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------- |
| `text` <PopoverInfo>Single text to animate when not using texts array.</PopoverInfo>                | ~string~                                                                                                                                                                      | <Na/>       |
| `texts` <PopoverInfo>Array of texts to cycle through for multiple typing sequences.</PopoverInfo>   | ~string[]~                                                                                                                                                                    | <Na/>       |
| `speed` <PopoverInfo>Typing speed in milliseconds between each character.</PopoverInfo>             | ~number~                                                                                                                                                                      | ~100~       |
| `delay` <PopoverInfo>Delay before starting animation in milliseconds.</PopoverInfo>                 | ~number~                                                                                                                                                                      | ~0~         |
| `showCursor` <PopoverInfo>Whether to display the blinking cursor.</PopoverInfo>                     | ~boolean~                                                                                                                                                                     | ~true~      |
| `cursor` <PopoverInfo>Character to use for the cursor.</PopoverInfo>                                | ~string~                                                                                                                                                                      | ~"\|"~      |
| `cursorClassName` <PopoverInfo>CSS class for styling the cursor element.</PopoverInfo>              | ~string~                                                                                                                                                                      | <Na/>       |
| `loop` <PopoverInfo>Whether to loop through texts array continuously.</PopoverInfo>                 | ~boolean~                                                                                                                                                                     | ~false~     |
| `pauseDuration` <PopoverInfo>Pause duration between loops in milliseconds.</PopoverInfo>            | ~number~                                                                                                                                                                      | ~2000~      |
| `className` <PopoverInfo>CSS class for the container element.</PopoverInfo>                         | ~string~                                                                                                                                                                      | <Na/>       |
| `onComplete` <PopoverInfo>Callback function when typing animation completes.</PopoverInfo>          | ~() => void~                                                                                                                                                                  | <Na/>       |
| `startOnView` <PopoverInfo>Whether to start animation when component enters viewport.</PopoverInfo> | ~boolean~                                                                                                                                                                     | ~true~      |
| `once` <PopoverInfo>Whether to animate only once when using startOnView.</PopoverInfo>              | ~boolean~                                                                                                                                                                     | ~false~     |
| `animation` <PopoverInfo>Animation preset variant for the text container.</PopoverInfo>             | ~enum~ <PopoverInfo> "fadeIn" \| "blurIn" \| "blurInUp" \| "blurInDown" \| "slideUp" \| "slideDown" \| "slideLeft" \| "slideRight" \| "scaleUp" \| "scaleDown" </PopoverInfo> | <Na/>       |
| `inViewMargin` <PopoverInfo>Margin for in-view detection trigger area.</PopoverInfo>                | ~string~                                                                                                                                                                      | <Na/>       |
