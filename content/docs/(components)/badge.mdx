## API Reference

This component provides a customizable badge system with flexible variants, sizes, and appearances.

### Badge

| **Prop**                                                                                               | **Type**                                                                                                                      | **Default** |
| ------------------------------------------------------------------------------------------------------ | ----------------------------------------------------------------------------------------------------------------------------- | ----------- |
| `variant` <PopoverInfo>Specifies the badge's theme.</PopoverInfo>                                      | ~enum~ <PopoverInfo> "primary" \| "secondary" \| "outline" \|"success" \| "warning" \| "info" \| "destructive" </PopoverInfo> | `secondary` |
| `appearance` <PopoverInfo>Determines the badge's visual appearance.</PopoverInfo>                      | ~enum~ <PopoverInfo> "default" \| "outline" \| "light" \| "ghost" </PopoverInfo>                                              | `default`   |
| `size` <PopoverInfo>Sets the size of the badge.</PopoverInfo>                                          | ~enum~ <PopoverInfo> "lg" \| "md" \| "sm" \| "xs" </PopoverInfo>                                                              | `md`        |
| `shape` <PopoverInfo>Determines the shape of the badge.</PopoverInfo>                                  | ~enum~ <PopoverInfo> "default" \| "circle" </PopoverInfo>                                                                     | `default`   |
| `disabled` <PopoverInfo>Specifies whether the badge is disabled.</PopoverInfo>                         | ~boolean~                                                                                                                     | `false`     |
| `asChild` <PopoverInfo>Renders the badge as a child element, using the `Slot` component.</PopoverInfo> | ~boolean~                                                                                                                     | `false`     |
| `className` <PopoverInfo>Additional class names for styling the badge.</PopoverInfo>                   | ~string~                                                                                                                      | <Na/>       |
