## API Reference

This component is built using [Motion](https://motion.dev/) primitives and includes the following custom props:

### SlidingNumber

| Prop                                                                                              | Type       | Default |
| ------------------------------------------------------------------------------------------------- | ---------- | ------- |
| `from` <PopoverInfo>The starting number value for the animation</PopoverInfo>                     | ~number~   | <Na/>   |
| `to` <PopoverInfo>The target number value for the animation</PopoverInfo>                         | ~number~   | <Na/>   |
| `duration` <PopoverInfo>Animation duration in seconds</PopoverInfo>                               | ~number~   | ~2~     |
| `delay` <PopoverInfo>Delay before animation starts in seconds</PopoverInfo>                       | ~number~   | ~0~     |
| `startOnView` <PopoverInfo>Whether to start animation when element comes into view</PopoverInfo>  | ~boolean~  | ~true~  |
| `once` <PopoverInfo>Whether to animate only once when in view</PopoverInfo>                       | ~boolean~  | ~false~ |
| `className` <PopoverInfo>Additional CSS classes to apply to the component</PopoverInfo>           | ~string~   | <Na/>   |
| `onComplete` <PopoverInfo>Callback function called when animation completes</PopoverInfo>         | ~function~ | <Na/>   |
| `digitHeight` <PopoverInfo>Height of each digit in pixels for the sliding animation</PopoverInfo> | ~number~   | ~40~    |
