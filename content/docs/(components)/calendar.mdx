## API Reference

This component provides a customizable calendar system built on top of the `react-day-picker` library.

### Calendar

This component is based on the [DayPicker](https://daypicker.dev) component and extends its functionality with custom styling and navigation controls.

| **Prop**                                                                                               | **Type**                  | **Default** |
| ------------------------------------------------------------------------------------------------------ | ------------------------- | ----------- |
| `className` <PopoverInfo>Additional class names for styling the calendar container.</PopoverInfo>      | ~string~                  | <Na/>       |
| `classNames` <PopoverInfo>Custom class names to override specific parts of the calendar.</PopoverInfo> | ~Record\<string, string>~ | <Na/>       |
| `showOutsideDays` <PopoverInfo>Determines whether to show days from adjacent months.</PopoverInfo>     | ~boolean~                 | ~true~      |
