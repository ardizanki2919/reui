## API Reference

This component is built using [Motion](https://motion.dev/) primitives and includes the following custom props:

### SVGText

| **Prop**                                                                                          | **Type**           | **Default** |
| ------------------------------------------------------------------------------------------------- | ------------------ | ----------- |
| `svg` <PopoverInfo>SVG content to display inside the text mask.</PopoverInfo>                     | ~ReactNode~        | <Na/>       |
| `children` <PopoverInfo>Text content to be masked and displayed.</PopoverInfo>                    | ~ReactNode~        | <Na/>       |
| `className` <PopoverInfo>CSS class for the container element.</PopoverInfo>                       | ~string~           | <Na/>       |
| `fontSize` <PopoverInfo>Font size for the text mask in CSS units or viewport width.</PopoverInfo> | ~string \| number~ | ~"20vw"~    |
| `fontWeight` <PopoverInfo>Font weight for the text mask.</PopoverInfo>                            | ~string \| number~ | ~"bold"~    |
| `as` <PopoverInfo>HTML element type to render for the container.</PopoverInfo>                    | ~ElementType~      | ~"div"~     |
