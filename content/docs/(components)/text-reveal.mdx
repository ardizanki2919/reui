## API Reference

This component is built using [Motion](https://motion.dev/) primitives and includes the following custom props:

### TextReveal

| **Prop**                                                                                            | **Type**                                                                                                                                                                                   | **Default** |
| --------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ----------- |
| `children` <PopoverInfo>Text content to be revealed with animation.</PopoverInfo>                   | ~string~                                                                                                                                                                                   | <Na/>       |
| `variant` <PopoverInfo>Animation style for text reveal effect.</PopoverInfo>                        | ~enum~ <PopoverInfo> "fade" \| "slideUp" \| "slideDown" \| "slideLeft" \| "slideRight" \| "scale" \| "blur" \| "typewriter" \| "wave" \| "stagger" \| "rotate" \| "elastic" </PopoverInfo> | ~"fade"~    |
| `className` <PopoverInfo>CSS class for the container element.</PopoverInfo>                         | ~string~                                                                                                                                                                                   | <Na/>       |
| `style` <PopoverInfo>Inline styles for the container element.</PopoverInfo>                         | ~React.CSSProperties~                                                                                                                                                                      | <Na/>       |
| `delay` <PopoverInfo>Delay before starting animation in seconds.</PopoverInfo>                      | ~number~                                                                                                                                                                                   | ~0~         |
| `duration` <PopoverInfo>Animation duration in seconds.</PopoverInfo>                                | ~number~                                                                                                                                                                                   | ~0.6~       |
| `staggerDelay` <PopoverInfo>Delay between individual character animations in seconds.</PopoverInfo> | ~number~                                                                                                                                                                                   | ~0.03~      |
| `once` <PopoverInfo>Whether to animate only once when using startOnView.</PopoverInfo>              | ~boolean~                                                                                                                                                                                  | ~true~      |
| `startOnView` <PopoverInfo>Whether to start animation when component enters viewport.</PopoverInfo> | ~boolean~                                                                                                                                                                                  | ~true~      |
| `wordLevel` <PopoverInfo>Whether to animate at word level instead of character level.</PopoverInfo> | ~boolean~                                                                                                                                                                                  | ~false~     |
| `onComplete` <PopoverInfo>Callback function when reveal animation completes.</PopoverInfo>          | ~() => void~                                                                                                                                                                               | <Na/>       |
