## API Reference

This component is built using [Radix UI Switch](https://www.radix-ui.com/primitives/docs/components/switch) primitives. For detailed information, please visit the full API reference.

### Switch

This component is based on the [Switch.Root](https://www.radix-ui.com/primitives/docs/components/switch#root) primitive and includes the following custom props:

| **Prop**                                                                                  | **Type**                                                                 | **Default** |
| ----------------------------------------------------------------------------------------- | ------------------------------------------------------------------------ | ----------- |
| `shape` <PopoverInfo>The shape of the switch.</PopoverInfo>                               | ~enum~ <PopoverInfo> "pill" \| "square" </PopoverInfo>                   | ~"pill"~    |
| `size` <PopoverInfo>The size of the switch.</PopoverInfo>                                 | ~enum~ <PopoverInfo> "xs" \| "sm" \| "md" \| "lg" \| "xl" </PopoverInfo> | ~"md"~      |
| `thumbClassName` <PopoverInfo>Additional class names for styling the thumb.</PopoverInfo> | ~string~                                                                 | ~undefined~ |

### SwitchWrapper

This is a custom component that provides context for permanent state and layout.

| **Prop**                                                                                                 | **Type**  | **Default** |
| -------------------------------------------------------------------------------------------------------- | --------- | ----------- |
| `permanent` <PopoverInfo>Whether the switch should maintain its appearance in both states.</PopoverInfo> | ~boolean~ | ~false~     |
| `className` <PopoverInfo>Additional class names for styling the wrapper.</PopoverInfo>                   | ~string~  | ~undefined~ |

### SwitchIndicator

This is a custom component that provides visual indicators for the switch states.

| **Prop**                                                                                 | **Type**                                          | **Default** |
| ---------------------------------------------------------------------------------------- | ------------------------------------------------- | ----------- |
| `state` <PopoverInfo>The state of the indicator.</PopoverInfo>                           | ~enum~ <PopoverInfo> "on" \| "off" </PopoverInfo> | ~"off"~     |
| `className` <PopoverInfo>Additional class names for styling the indicator.</PopoverInfo> | ~string~                                          | ~undefined~ |
