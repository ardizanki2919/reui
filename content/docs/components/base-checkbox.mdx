## API Reference

### BaseCheckbox

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `size` | ~enum~ <PopoverInfo> "sm" \| "md" \| "lg" </PopoverInfo> | ~"md"~ | The size of the checkbox |
| `className` | ~string~ | ~undefined~ | Additional CSS classes to apply |
| `checked` | ~boolean~ | ~undefined~ | Whether the checkbox is checked |
| `defaultChecked` | ~boolean~ | ~undefined~ | Whether the checkbox is checked by default |
| `indeterminate` | ~boolean~ | ~undefined~ | Whether the checkbox is in an indeterminate state |
| `disabled` | ~boolean~ | ~undefined~ | Whether the checkbox is disabled |
| `onCheckedChange` | ~function~ | ~undefined~ | Callback fired when the checked state changes |

The component also accepts all props from Base UI's Checkbox.Root component.
